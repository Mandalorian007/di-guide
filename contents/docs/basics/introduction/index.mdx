# Welcome to DI Guide

Welcome to **DI Guide**! This is a comprehensive resource hub for Diablo Immortal players, providing up-to-date information, strategies, and builds to help you navigate the world of Sanctuary.

## Community-Driven Resource

DI Guide is a **community-driven project** maintained by the ThunderClap Gaming Community. Our goal is to provide accurate, helpful information for players of all experience levels.

<Note title="Contribute"> 
 Interested in helping us improve? Check out our [GitHub repository](https://github.com/Mandalorian007/di-guide) to get started! From new guides to build suggestions, all contributions are welcome.
</Note>

## What You'll Find Here

**DI Guide** is designed to be your go-to resource for all things Diablo Immortal:

- **Class Builds:** Optimized builds for all classes in different content types
- **Game Mechanics:** Detailed explanations of complex game systems
- **Farming Guides:** Efficient routes and strategies for resource gathering
- **Event Information:** Updates on current and upcoming events
- **Dungeon Strategies:** Tips for completing dungeons and raids efficiently

### Content Categories

| **Category**                | **Description**                                          |
|-------------------------------|----------------------------------------------------------|
| Class Builds                  | Detailed guides for each class with PvE and PvP builds    |
| Dungeons & Raids              | Strategies for completing group content efficiently       |
| Gear & Items                  | Information on legendary items, set bonuses, and gems     |
| Leveling                      | Guides for efficient character progression               |
| Battlegrounds                 | PvP tactics and battleground strategies                  |
| Events                        | Information on seasonal and special events               |
| Advanced Mechanics            | In-depth analysis of game systems and hidden mechanics   |
